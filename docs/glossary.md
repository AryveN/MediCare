# Slovníček pojmů MediCare

- **Monorepo** – jeden repozitář obsahující více aplikací/balíčků.
- **Workspace** – logický balíček uvnitř monorepa spravovaný npm (má vlastní `package.json`).
- **Endpoint** – URL adresa, na kterou lze poslat HTTP požadavek.
- **DTO (Data Transfer Object)** – objekt popisující strukturu dat přenášených API.
- **Swagger** – nástroj generující dokumentaci k REST API.
- **Logger / Pino** – knihovna pro strukturované logování.
- **Rate-limit** – omezení počtu požadavků v čase, ochrana proti přetížení.
- **Helmet** – sada bezpečnostních HTTP hlaviček pro Express/Nest.
- **CORS** – Cross-Origin Resource Sharing, povolení přístupu z jiné domény/portu.
- **CI (Continuous Integration)** – automatizované ověřování kódu (lint, testy) na každé změně.
- **Devcontainer** – kontejnerová konfigurace pro VS Code, sjednocuje prostředí týmu.
- **.env** – soubor s citlivými nebo prostředím specifickými proměnnými.
